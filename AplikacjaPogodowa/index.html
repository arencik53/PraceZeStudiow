Informacje o PWA

Co to jest PWA?

Progressive Web App (PWA) to aplikacja internetowa, która wykorzystuje nowoczesne funkcje przeglądarek, aby zapewnić użytkownikom doświadczenie podobne do natywnych aplikacji.

Funkcje tej aplikacji PWA:

  • Możliwość zainstalowania na urządzeniu
  • Działanie offline (tryb offline)
  • Responsywny design
  • Dostęp do API urządzenia
  • Automatyczne aktualizacje

Jak zainstalować?

W zależności od przeglądarki i urządzenia, możesz zobaczyć przycisk "Zainstaluj" w pasku adresu lub w menu przeglądarki.
